News More Fun
-------------

A ruby on rails implementation of 
www.xkcd.com/1288


Installation
------------

* git clone git@github.com:AJFaraday/news-more-fun.git
* cd news-more-fun
* bundle install 
* rake db:create
* rake db:migrate
* rake db:seed
* rake rss:pull 
* rails c
* firefox localhost:3000
